# better-arch
 
bta is a tool that can quickly configure files.
- Install packages.
- Remove packages.
- Get the git source code in the specified directory.
- Remove git source
- coming soon...

## Running

#### Install with json
```sh
bta -i red-team-tools.json
```

#### Remove with json
```sh
bta -r red-team-tools.json
```
